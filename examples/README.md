# Examples

Place small sample sheets, toy inputs, and documented command examples here as
the prototype matures.

The checked-in sample sheets currently target the Control dataset that is
available under `data/` in this workspace:

- `pipeline_samples.csv`
- `pipeline_samples_test.csv`

If you add `drugA` or `drugB` FASTQ folders locally, you can extend these CSV
files with additional rows for those samples.
